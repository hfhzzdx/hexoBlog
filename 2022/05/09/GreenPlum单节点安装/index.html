GreenPlum单节点安装

GreenPlum GP
数据库 GP
发布日期:   2022-05-10

GreenPlum单节点安装

准备工作

1. 下载安装包

GP下载

2. 关闭SELINUX

修改/etc/selinux/config中 SELINUX的值为disabled

3. 关闭防火墙

[root@centos ~]# systemctl stop firewalld                    #关闭防火墙
[root@centos ~]# systemctl disable firewalld                 #设置防火墙开机不启动

4. 修改主机名hostname

修改hostname为gpnode

[root@centos ~]# cat /etc/hostname    #永久修改,重启生效
gpnode
[root@centos ~]# 
[root@centos ~]#  hostname gpnode     #临时修改
[root@centos ~]# hostname
gpnode
[root@centos ~]#

5. 创建gpadmin用户和用户组

gp数据库默认superuser为gpadmin

[root@centos ~]# useradd gpadmin
[root@centos ~]# passwd gpadmin
Changing password for user gpadmin.
New password: 
BAD PASSWORD: The password is shorter than 8 characters
Retype new password: 
passwd: all authentication tokens updated successfully.
[root@centos ~]#

6. 配置内核参数

[root@centos ~]# vim /etc/sysctl.conf  

kernel.shmmax = 500000000
kernel.shmmni = 4096
kernel.shmall = 4000000000
kernel.sem = 250 512000 100 2048
kernel.sysrq = 1
kernel.core_uses_pid = 1
kernel.msgmnb = 65536
kernel.msgmax = 65536
kernel.msgmni = 2048
net.ipv4.tcp_syncookies = 1
net.ipv4.conf.default.accept_source_route = 0
net.ipv4.tcp_tw_recycle = 1
net.ipv4.tcp_max_syn_backlog = 4096
net.ipv4.conf.all.arp_filter = 1
net.ipv4.ip_local_port_range = 10000 65535
net.core.netdev_max_backlog = 10000
net.core.rmem_max = 2097152
net.core.wmem_max = 2097152
vm.overcommit_memory = 2


# sysctl -p 生效或者重启生效
[root@centos ~]# sysctl -p
kernel.shmmax = 500000000
kernel.shmmni = 4096
 ...

7. 修改资源限制

[root@centos ~]# vim /etc/security/limits.conf  

* soft nofile 65536
* hard nofile 65536
* soft nproc 131072
* hard nproc 131072

创建instance所需目录

[root@gpnode greenplum-db]# mkdir -p /home/gpdata
[root@gpnode greenplum-db]# mkdir -p /home/gpdata/master
[root@gpnode greenplum-db]# mkdir -p /home/gpdata/gp1
[root@gpnode greenplum-db]# mkdir -p /home/gpdata/gp2
[root@gpnode greenplum-db]# mkdir -p /home/gpdata/gp3
[root@gpnode greenplum-db]# mkdir -p /home/gpdata/gp4
[root@gpnode greenplum-db]#

变更文件属组

[root@gpnode ~]# chown -R gpadmin.gpadmin /usr/local/greenplum-db*
[root@gpnode ~]# chown -R gpadmin:gpadmin /home/gpdata
[root@gpnode ~]# chown -R gpadmin:gpadmin /home/gpdata/master
[root@gpnode ~]# chown -R gpadmin:gpadmin /home/gpdata/gp*
[root@gpnode ~]#

配置hosts文件

echo "192.168.191.3	gpnode" >> /etc/hosts

切换gpadmin用户

[root@gpnode ~]# su - gpadmin
-bash-4.2$

编辑gpadmin环境变量

-bash-4.2$ cat .bash_profile 
source /usr/local/greenplum-db-6.2.1/greenplum_path.sh
export MASTER_DATA_DIRECTORY=/home/gpdata/master/gpseg-1
export PGPORT=2345
export PGUSER=gpadmin
export PGDATABASE=gpdb

-bash-4.2$ source .bash_profile      #使环境变量生效

设置instance的节点

-bash-4.2$ ll
total 0

-bash-4.2$ cat all_hosts_file  
gpnode
-bash-4.2$

[gpadmin@gpnode ~]$ source /usr/local/greenplum-db-6.2.1/greenplum_path.sh
[gpadmin@gpnode ~]$ python2.7 /usr/local/greenplum-db/bin/gpssh-exkeys -f ~/all_hosts_file

这里需要特别*注意,由于之前先设置的环境变量再配置的互通,导致互通一直失败.*

-bash-4.2$ python2.7 /usr/local/greenplum-db/bin/gpssh-exkeys -f ~/all_hosts_file 
[STEP 1 of 5] create local ID and authorize on local host
[ERROR gpnode] authentication check failed:
     Warning: Permanently added the RSA host key for IP address 'fe80::fcdf:bc2:c366:b936%ens33' to the list of known hosts.
     Permission denied (publickey,gssapi-keyex,gssapi-with-mic,password).
[ERROR] cannot establish ssh access into the local host

最后解决方案,删除用户和组,先配置免密,再设置环境变量

编辑GreenPlum初始化文件

[gpadmin@gpnode ~]$ cp /usr/local/greenplum-db/docs/cli_help/gpconfigs/gpinitsystem_config ~  #gpinitsystem_config为自带模板,拷贝到家目录进行编辑
[gpadmin@gpnode ~]$ 
修改gpinitsystem_config配置文件为如下
SEG_PREFIX=gpseg
PORT_BASE=33000
declare -a DATA_DIRECTORY=(/home/gpdata/gp1 /home/gpdata/gp2 /home/gpdata/gp3 /home/gpdata/gp4)
MASTER_HOSTNAME=gpnode
MASTER_PORT=2345
MASTER_DIRECTORY=/home/gpdata/master
DATABASE_NAME=gpdb
(注:ENCODING​=​UNICODE 此字符集一定要选择UNICODE)

设置节点服务器

[gpadmin@gpnode ~]$ echo "gpnode" > seg_hosts_file
[gpadmin@gpnode ~]$

使用GP数据库

启动和停止

启动GP

[gpadmin@gpnode ~]$ gpstart

停止GP

[gpadmin@gpnode ~]$ gpstop

查看服务状态

[gpadmin@gpnode ~]$ gpstate

快速关闭

[gpadmin@gpnode ~]$ gpstop -M fast

重启

[gpadmin@gpnode ~]$ gpstop r
