3. 建表语句

testDb=# create table test001(id int,name varchar(128));           --id 为分布键
NOTICE:  Table doesn't have 'DISTRIBUTED BY' clause -- Using column named 'id' as the Greenplum Database data distribution key for this table.
HINT:  The 'DISTRIBUTED BY' clause determines the distribution of data. Make sure column(s) chosen are the optimal data distribution key to minimize skew.
CREATE TABLE


testDb=# \d
              List of relations
 Schema |  Name   | Type  |  Owner  | Storage 
--------+---------+-------+---------+---------
 public | test001 | table | gpadmin | heap
(1 row)


testDb=# create table test002(id int ,name varchar(128)) distributed by(id);    -- id为分布键
CREATE TABLE
testDb=# 


testDb=# create table test003(id int ,name varchar(128)) distributed by(id,name);  --id和name为复合分布键
CREATE TABLE
testDb=# 

testDb=# create table test004(id int ,name varchar(128)) distributed randomly;   --使用随机分布
CREATE TABLE
testDb=# 

testDb=# create table testlike(like test001);                     -- 创建一模一样的表
NOTICE:  table doesn't have 'DISTRIBUTED BY' clause, defaulting to distribution columns from LIKE table
CREATE TABLE
testDb=#

4. 插入语句

执行insert语句注意分布键不要为空,否则分布键默认变成null’,数据都被保存到一个节点上会导致分布不均

testDb=# insert into test001 values (100,'tom'),(101,'lily'),(102,'jack'),(103,'linda');
INSERT 0 4

testDb=# insert into test002 values (200,'tom'),(101,'lily'),(202,'jack'),(103,'linda');
INSERT 0 4
testDb=#

5. 更新语句

不能批量对分布键执行update,因为分布键执行update需要将数据重分布.

update test002 set id=203 where id=202;

6. 删除

delete 删除整张表比较慢,所以建议使用truncate

testDb=# truncate table test002;
TRUNCATE TABLE
testDb=# select * from test002;
 id | name 
----+------
(0 rows)

二、常用数据类型

类型 存储空间 描述 范围 时间精度
数值 smallint 2字节 小范围整数 -32768 ~~~ +32768
integer 4字节 常用的整数 -2147483648 ~~~ +2147483647
bigint 8字节 大范围整数 -9223372036854775808 ~~~ ∞
decimal 变长 用户自定义精度,精确 无限制
numeric 变长 用户自定义精度,精确 无限制
real 4字节 变精度,不精确 6位十进制数字精度
double precision 8字节 变精度,不精确 15位十进制数字精度
serial 4字节 自增整数 1 ~~~ +2147483647
bigserial 8字节 大范围的自增整数 1 ~~~ +9223372036854775807
字符 character varying(n) varchar(n) 变长,有长度限制
character(n) char(n) 定长,不足补空白
text 变长,无长度限制
时间 timestamp[(p)][without time zone] 8字节 日期和时间 4713BC ~~~ 5874897AD 1毫秒
timestamp[(p)][with time zone] 8字节 日期和时间带时区 4713BC ~~~ 5874897AD 1毫秒
interval[(p)] 12字节 时间间隔 -178000000年 ~~~ +178000000年 1毫秒
date 4字节 只用于表示日期 4713BC ~~~ 5874897AD 1天
time[(p)][without time zone] 8字节 只用于表示一日内时间 0:00:00 ~~~ 24:00:00 1毫秒
time[(p)][with time zone] 12字节 只用于表示一日内时间带时区 0:00:00+1459 ~~~ 24:00:00-1459 1毫秒

三、常用函数

1. 字符串常用函数

函数 返回类型 描述 例子 结果
string || string text 字符串连接 ‘aa’ || ‘bb’ aabb
length(string) int string中字符的数目 length(‘aaa’) 3
position(substring in string) int 指定的子字符串的位置 position(‘om’ in ‘Thomas’) 3
substring(string [from int] [for int]) text 抽取子字符串 substring(‘Thomas’ from 2 to 3) hom
trim([leading|trailing|both] [characters] from string) text 从字符串string的开头/结尾/两边删除只包含characters中字符(默认是一个空白)的最长的字符串 trim(both ‘x’ from ‘xTomxx’) Tom
lower(string) text 字符串转小写 lower(‘TOM’) tom
upper(string) text 字符串转大写 upper(‘tom’) TOM
overlay(string placing string from int [for int]) text 替换字符串 overlay(‘Txxxxas’ placing ‘hom’ from 2 for 4) Thomas
replace(string text,from tesxt, to text) text 把字符串string中出现的所有子字符串from替换成子字符串to replace(‘abcdefabcdef’, ‘cd’, ‘XX’) abXXefabXXef
split_part(string text, delimiter text, field int) text 根据delimiter分割字符串返回生成的第field个子字符串(1为基准) split_part(‘abc|def|ghi’, ‘|’, 2) def

2. 时间函数

函数 返回类型 描述 例子 结果
age(timestamp, timestamp) interval 减去参数后的”符号化”结果 age(timestamp’2001-04-10’, timestamp’1957-06-13’ 43 years 9 mons 27 days
age(timstamp) interval 从current_date减去参数中的日期 age(timestamp’1957-06-13’) 64 years 10 mons 27 days
current_date date 当前的日期 2022-05-10
current_time time with time zone (timetz) 当日时间 16:57:43.65079+08
current_timestamp timestamp with time zone (now) 当前事务开始的时间戳 2022-05-10 16:59:24.739026+08
date_part(text,timestamp) double precision 获取子域(等效于extract) date_part(‘hour’,timestamp’2001-02-16 20:38:40’) 20
date_trunc(text, timestamp) timestamp 截断成指定的精度 date_trunc(‘hour’, timestamp’2001-02-16 20:38:40’) 2001-02-16 20:00:00
extract(field from timestamp) double precision 获取子域 extract(‘hour’ from timestamp’2001-02-16 20:38:40’) 20
now() timestamp with time zone 当前事务开始的时间戳 2022-05-10 17:06:23.463615+08

字符串列转行 string_agg

string_agg(str,symbol [order by str])
（按照某字段排序）将str列转行，以symbol分隔
testDb=# select * from test001;
 id  | name  
-----+-------
 100 | tom
 100 | jack
 101 | lily
 102 | jack
 103 | linda
(5 rows)

testDb=# select id,string_agg(name,'|') as res from test001 group by id;
 id  |   res    
-----+----------
 100 | tom|jack
 101 | lily
 102 | jack
 103 | linda
(4 rows)


testDb=# select id,string_agg(name,'|' order by name desc) as res from test001 group by id;
 id  |   res    
-----+----------
 101 | lily
 102 | jack
 103 | linda
 100 | tom|jack
(4 rows)
